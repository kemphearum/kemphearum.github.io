# Architecture Decision Records

This folder holds portfolio ADRs.

Create an ADR when a decision changes or freezes a meaningful architecture choice, such as routing strategy, Firebase data shape, admin registry behavior, RBAC enforcement, deployment topology, dependency adoption, or major UX platform direction.

Suggested naming: `ADR-001-short-title.md`.
