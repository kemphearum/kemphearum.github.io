# Admin Entities

## Entities

Admin entities include users, role permissions, sessions, audit logs, messages, settings, feature flags, analytics visits, search/download events, backup/restore payloads, and notification history.

## Behavior

Admin entities support role-aware viewing, mutating, auditing, import/export, restore, and status management where implemented.
