# Contact, Auth Log, Geo, and DB Sync APIs

## Contact

`/api/contact` receives public contact submissions and persists validated messages.

## Auth Log

`/api/auth-log` records authentication-related audit events through server-side logic.

## Geo

`/api/geo` proxies geolocation lookups so API keys are not client-bundled.

## DB Sync

`/api/db-sync` supports database administration workflows through server logic and admin controls.
