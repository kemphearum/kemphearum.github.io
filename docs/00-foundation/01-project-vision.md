# Project Vision

## Vision

Provide a professional cybersecurity and governance portfolio that showcases practical ISO/IEC 27001:2022, ISMS, risk management, secure SDLC, and compliance experience while also operating as a private CMS.

## Objectives

- Present credible public content for recruiters, auditors, and technology stakeholders.
- Manage portfolio content through a secure, role-aware admin CMS.
- Preserve multilingual English and Khmer support.
- Stay deployable on free-tier Vercel, Firebase Spark, GitHub Pages, and GitHub Actions where applicable.

## Success Criteria

The site is fast, accessible, SEO-friendly, maintainable, secure, and easy to update without changing source code for ordinary content edits.
