# Repository Structure

## Key Folders

- `app/`: React Router route table, route modules, SSR/API route modules, sitemap, RSS, robots, and fallback routes.
- `src/pages/`: public route-level pages and the admin CMS.
- `src/sections/`: public portfolio sections and section-level UI.
- `src/domain/`: content normalizers, status logic, slugs, and pure domain behavior.
- `src/services/`: Firestore, auth, content, analytics, message, settings, and database services.
- `src/server/`: backend-only logic used by API route modules.
- `src/registry/`: content type, navigation, feature, permission, settings, search, and dashboard registries.
- `src/shared/`: reusable shared UI components.
- `docs/`: versioning plus this phase-based documentation program.
