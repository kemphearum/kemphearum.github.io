# Technology Stack Baseline

## Runtime Stack

- React 19
- React Router 7
- Vite 7
- Firebase 12, Firestore, and Firebase Authentication
- Vercel React Router adapter
- TanStack Query 5
- SCSS and CSS Modules

## Supporting Libraries

The app uses existing dependencies such as Radix UI, lucide-react, framer-motion, react-markdown, react-syntax-highlighter, qrcode.react, recharts, axios, and Firebase Admin for server-side routes.

## Constraint

Do not introduce new dependencies unless a future task explicitly approves the dependency and explains why existing libraries cannot meet the need.
