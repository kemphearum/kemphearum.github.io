# i18n Module

## Sources

- `src/i18n/en.json`
- `src/i18n/km.json`
- `src/context/LanguageContext.jsx`
- `src/hooks/useTranslation.js`
- `src/utils/localization.js`

## Behavior

The app supports English and Khmer UI/content patterns, language persistence, and localized field selection for content-bearing sections.

## Rule

New content and admin UI should use existing translation and localized-field helpers.
