# Indexes

## Source

`firestore.indexes.json` defines composite indexes.

## Indexed Areas

Indexes cover public content visibility and sort queries, title/company sorting, logs and audit filters, messages by sender email, and status/listing queries for skills, certificates, education, awards, publications, and speaking.

## Maintenance Rule

When a service adds a new compound query, update `firestore.indexes.json` and this document together.
