# Seed and Sample Data

## Seed Command

`npm run seed -- --project kem-phearum` runs `scripts/seed-sample-data.mjs` according to `package.json` and `README.md`.

## Sample Data Sources

Sample data and helpers are present under `src/utils/sampleData.js` and related migration/seed scripts.

## Constraint

Do not seed or mutate Firestore as part of documentation work.
