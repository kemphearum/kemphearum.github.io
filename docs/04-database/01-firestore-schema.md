# Firestore Schema

## Overview

Firestore is the primary application database. Client services and server modules read and write content, admin, analytics, audit, user, message, and settings documents.

## Known Collections

Observed collection groups and collections include posts, projects, logs, auditLogs, experience, messages, skills, certificates, education, awards, publications, speaking, users, rolePermissions, visits, settings, and feature flags.
