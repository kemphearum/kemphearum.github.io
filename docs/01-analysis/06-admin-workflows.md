# Admin Workflows

## Core Workflows

- Authenticate and enter the `/admin` shell.
- Navigate registered admin tabs through the sidebar and command palette.
- Search, filter, select, create, edit, publish, feature, import, export, restore, or delete content according to permissions.
- Review messages, analytics, audit logs, database status, and user permissions.

## Shared Workflow Machinery

Content CRUD tabs use `useAdminCrud.js`, `adminCrudIO.js`, `BulkActionsBar.jsx`, `DataTable.jsx`, and registered feature descriptors.
