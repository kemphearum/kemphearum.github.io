# Integration Points

## Internal Integrations

- React Router route modules integrate pages and server API handlers.
- Services integrate client-side features with Firestore.
- Server modules integrate API routes with Firebase Admin behavior.
- Registries integrate admin navigation, permissions, search, settings, content types, and dashboard widgets.

## External Services

- Firebase Authentication and Firestore.
- Vercel SSR/API runtime and analytics/speed insights packages.
- Optional geolocation API through server-side proxying.
