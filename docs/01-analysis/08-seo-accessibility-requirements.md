# SEO and Accessibility Requirements

## SEO Requirements

- Keep sitemap, RSS, robots, canonical URLs, route metadata, and blog/project detail metadata functional.
- Preserve public content slugs and redirect behavior.
- Ensure CMS content changes do not break route discovery or feed generation.

## Accessibility Requirements

- Preserve keyboard access for admin tables, command palette, forms, dialogs, and public navigation.
- Use semantic headings and landmarks on public pages.
- Ensure responsive layouts work for desktop, tablet, and mobile.
