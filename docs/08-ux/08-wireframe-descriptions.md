# Wireframe Descriptions

## Public Home

The public home page is a vertical portfolio experience composed of sections such as hero, about, credibility, experience, projects, skills, certificates, education, recruiter snapshot, contact, and footer.

## Projects and Blog

Listing pages support scanning and navigation to detail pages. Detail pages emphasize readable content, related entries, and SEO-friendly structure.

## Admin

The admin shell uses a sidebar, header, content frame, cards, tables, dialogs, toolbars, forms, notification center, and command palette.
