# Admin Access Matrix

| Area | Typical Access | Source |
|---|---|---|
| Dashboard | Authenticated admin roles based on visible widgets | Dashboard registry and admin README |
| Blog and projects | Admin plus editor/writer patterns | Firestore rules and permissions |
| User management | Admin/superadmin controlled | User management components and rules |
| Audit logs | Superadmin-sensitive access | Admin README and rules |
| Messages | Admin or custom role with messages tab | Rules helper functions |
| Database | Admin-controlled operational module | Database tab and services |
