# Security Policy

## Supported Versions

Currently, only the `main` branch (and the latest stable release) receives security updates.

| Version | Supported          |
| ------- | ------------------ |
| >= 1.0  | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please **do not** create a public issue. 

Instead, report it privately to the repository owner. We will review the vulnerability and provide a timeline for resolution. All security reports are treated seriously, and we aim to address them promptly to ensure the safety and integrity of the application.

1. Email the repository owner (contact info available on the public site or profile).
2. Wait for an acknowledgment (usually within 48 hours).
3. A fix will be developed and released as quickly as possible.
